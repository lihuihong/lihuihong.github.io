HtmlUnit初次使用

发表于 2018-01-22 | 分类于 开发 |
字数统计: 1,078 字 | 阅读时长 ≈ 5 分钟 
一.htmlunit简介

Jsoup只能爬去静态页面，htmllunit相比于Jsoup可以模拟浏览器运行，可以抓取动态页面包括js操作，是一个强大的无界面Java浏览器，是junit的扩展之一，它提供了丰富的API，在读取页面后，您可以有效的使用htmlunit分析页面上的内容，并调用页面、填写表单、点击链接，就像您在“正常”浏览器中所作的一样。它具有相当好的JavaScript支持，并且能够使用相当复杂的AJAX库来工作，根据所使用的配置模拟Chrome、Firefox或Internet Explorer。HtmlUnit还可以用来进行页面的测试工作，实现网页自动化 测试。

二.htmlunit使用
  1. 需要引入jar包 htmlunit-2.29-OSG.jar
    http://download.csdn.net/download/qq_26420743/9811028
  2. 功能特点:

    支持HTTP和HTTPS协议

    支持Cookie

    能够指定服务器的失败响应是否应该抛出异常，或者应该返回为适当类型的页面（基于内容类型）

    支持提交方法POST和GET（以及HEAD，DELETE，…）

    能够自定义要发送到服务器的请求头

    支持HTML响应

    HTML页面的包装器，可以轻松访问其中包含的所有信息

    支持提交表单

    支持点击链接

    支持步行HTML文档的DOM模型

    代理服务器支持

    支持基本和NTLM身份验证

    优秀的JavaScript支持

    支持将拿到的网页转成Document对象
  3. 具体使用：

    既然是爬虫，当然就是要抓取数据，Java最常用的是Jsoup，但是它只能抓取静态网页的数据，对于Js，Ajax加载的动态网页是抓取不到数据的，这个时候，就需要用HtnlUnit这个强大的工具了。

-爬取网页并转化为Document对象：

WebClient  wc = new WebClient(BrowserVersion.CHROME);//模拟chrome浏览器
wc.getOptions().setUseInsecureSSL(true);
wc.getOptions().setJavaScriptEnabled(true); // 启用JS解释器，默认为true
wc.getOptions().setCssEnabled(false); // 禁用css支持
wc.getOptions().setThrowExceptionOnScriptError(false); // js运行错误时，是否抛出异常
wc.getOptions().setTimeout(100000); // 设置连接超时时间 ，这里是10S。如果为0，则无限期等待
wc.getOptions().setDoNotTrackEnabled(false);
wc.setAjaxController(new NicelyResynchronizingAjaxController());// 设置Ajax异步
HtmlPage page = wc.getPage("http://36kr.com/");//获取页面
String pageXml = page.asXml();
Document doc = Jsoup.parse(pageXml);// jsoup解析文档

爬取数据操作：

//拿到标题，文章地址，图片地址(首页相关信息)
public static List<AdHeadBean> getHomeBeans(Document doc) {
    AdHeadBean bean = new AdHeadBean();
    List<AdHeadBean> adHeadBeans = new ArrayList<AdHeadBean>();

    elements = doc.select("div.kr_article_list").select("ul.feed_ul");

    Elements elements1 = elements.select("div.am-cf");
    //System.out.println(elements1);
    for (Element element : elements1) {//select("div data-stat-click").attr("abs:href");
        img = ImageUtils.getCutImageUrl(element.select("a").select("img").attr("data-src"));
        if (img.equals("")){
            System.out.println("没有数据");
        }else {
            String mark = element.select("a").select("span.mark").text();
            String href = "http://36kr.com" + element.select("a").attr("href");
            String title = element.getElementsByTag("h3").text();
            String content = element.select("a").select("div.abstract").text();
            bean.setContent(content);
            String avthor = element.select("div.user-info").select("a").text();
            String time = element.select("div.time-div").select("span").first().text();

            System.out.println("======" + avthor + "====" + time);
            adHeadBeans.add(new AdHeadBean(title,img,href,content,mark,avthor,time));
        }

        System.out.println("======" + img);
    }

    return adHeadBeans;
}
